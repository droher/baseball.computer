# SQLMesh documentation URL index

Canonical base: `https://sqlmesh.readthedocs.io/en/stable/`

When in doubt, fetch from `/en/stable/` — it tracks the latest tagged
release. `/en/latest/` tracks `main` and may be ahead of what users have
installed.
